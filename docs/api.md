## Loading 加载组件

加载数据时显示动效。

## 代码演示

### API

|参数|说明|类型|默认值|
|:---|:----|:----|:------|
|loadingType|类型(`rotate` `line`)|string|rotate|
|size|加载大小(`large` `medium` `small`)|string|medium|
|color|颜色(`primary` `success``warning`)|string|''|
|container|渲染到的容器|node|body|
|showBackDrop|是否显示遮罩|boolean|true|
