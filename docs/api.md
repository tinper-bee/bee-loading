## Loading 加载组件

加载数据时显示动效。

## 代码演示

### API

|参数|说明|类型|默认值|
|:---|:----|:----|:------|
|loadingType|类型(`rotate` `line`)|string|''|
|size|加载大小(`large` `medium` `small`)|string|medium|
|colors|颜色(`primary` `success``warning`)|string|''|
|backColor|背景颜色(`light` `dark`)|string|'light'|
|describe|是否带有文字||boolean|false|
